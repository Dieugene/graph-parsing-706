# Регистрация ценных бумаг - регуляторный граф

Парсинг положения "О стандартах эмиссии ценных бумаг" Банка России, и связанных документов. Формирование графа зависимостей "регистрационные действия - требуемые документы - требуемые поля".

## Цель

Решаем проблему сложности восприятия массива регуляторных требований, подготовки фундамента для последующего упрощения и оптимизации регулирования.

## Структура проекта

```
.agents/          # AI-агенты для разработки
00_docs/          # Документация и стандарты
01_tasks/         # Задачи разработки
02_src/           # Исходный код
03_data/          # Данные (исходные документы, результаты)
04_logs/          # Логи работы
```

## Разработка

Проект использует AI-агентов для организации разработки. Подробности в `DEVELOPER_WORKFLOW.md`.
